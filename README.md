# Лабораторная работа №1. Создание шаблона для отображения записи блога

## Запуск локально

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

Открыть: `http://127.0.0.1:5000`

## Возможности

- Страница списка постов (`/posts`) с карточками, превью текста и ссылкой на полный пост.
- Страница поста (`/posts/<index>`): заголовок, автор, дата, изображение, текст, форма комментария и вывод комментариев с ответами.
- Общий шаблон `templates/base.html` с навигацией и подвалом (footer).
- Данные (посты/комментарии) генерируются при запуске с помощью `Faker` (без базы данных).

## Маршруты

- `GET /` — страница задания (`templates/index.html`)
- `GET /posts` — список постов (`templates/posts.html`)
- `GET /posts/<int:index>` — конкретный пост (`templates/post.html`)
- `GET /about` — страница “Об авторе” (`templates/about.html`)

## Структура проекта

- `app.py` — Flask‑приложение (маршруты + генерация данных)
- `templates/` — HTML‑шаблоны (Jinja2)
  - `base.html` — базовый шаблон (header/footer)
  - `posts.html` — список постов
  - `post.html` — страница одного поста
- `static/` — статика
  - `styles.css` — дополнительные стили
  - `images/` — изображения постов, `avatar.jpg`, пример страницы
